# Lambda (lambda)
Lambda gives your team instant access to the compute you need to build and deploy Artificial Intelligence. Lambda Cloud provides on-demand GPU instances powered by NVIDIA A100, H100, and other high-performance GPUs for deep learning training and inference workloads.

## APIs

### Lambda Cloud API
The Lambda Cloud API allows you to manage GPU cloud instances programmatically. You can launch, list, restart, and terminate instances, manage SSH keys, list available instance types and images, and manage persistent storage file systems through a RESTful interface.

**Human URL:** [https://docs.lambda.ai/public-cloud/cloud-api/](https://docs.lambda.ai/public-cloud/cloud-api/)

**Base URL:** https://cloud.lambdalabs.com/api/v1
